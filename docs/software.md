# Software

## OSX/macOS

When getting a new computer, you can use [druid-toolbelt](https://github.com/druidfi/druid-toolbelt) to install most common tools: 

```
$ sh -c "$(curl -fsSL https://github.com/druidfi/druid-toolbelt/raw/master/install.sh)"
```
