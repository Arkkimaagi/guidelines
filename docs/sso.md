# SSO (Single-Sign-On) with PHP

How to use SSO e.g. with Microsoft AD/Azure AD and Google.

## Terminology

- IdP = Identity Provider e.g. AD
- SP = Service Provider e.g. Drupal

## Requirements

- [SimpleSAMLphp](https://simplesamlphp.org/)
